# Privacy Policy

**DeDupe** - Last updated: February 2026

## Overview

DeDupe is a desktop application that runs entirely offline on your local machine. Your images and data never leave your device.

## Data Collection

**DeDupe does not collect, transmit, or store any personal data on external servers.**

Specifically:

- **No image uploads.** All image processing, feature extraction, and similarity analysis happen locally on your device. No images or image data are sent to any server, cloud service, or third party.
- **No telemetry.** DeDupe does not collect usage statistics, analytics, or crash reports.
- **No network requests.** DeDupe does not require an internet connection to function and makes no outbound network requests during normal operation.
- **No user accounts.** DeDupe does not require registration or sign-in.

## Data Stored Locally

DeDupe may store the following data on your device:

- **Application settings and preferences**, stored in the app's local data folder managed by Windows.
- **Log files** for debugging purposes, stored in the app's local data folder. These logs may contain file paths of processed images but do not contain image content. Log files are not transmitted anywhere.
- **Temporarily processed images** for feature extraction and similarity analysis. These temporary files are stored in the app's local data folder and are deleted after.

All locally stored data is removed when you uninstall the application through Windows or the Microsoft Store.

## Image Metadata

DeDupe may read image metadata (such as file size, resolution, and format) to display information and assist with duplicate management. DeDupe does not read, process, or display EXIF data such as GPS coordinates, camera information, or timestamps unless explicitly shown in the application interface.

## Third-Party Services

DeDupe does not integrate with or transmit data to any third-party services, APIs, or analytics platforms.

## Microsoft Store

If you install DeDupe through the Microsoft Store, Microsoft may collect standard installation and usage telemetry as described in [Microsoft's Privacy Statement](https://privacy.microsoft.com/privacystatement). This data collection is handled by the Microsoft Store platform and is outside DeDupe's control.

## Open Source

DeDupe is open source software licensed under GPL v3. You can review the complete source code at [GitHub](https://github.com/Jan-Eric-B/DeDupe) to verify these privacy practices.

## Changes to This Policy

If this privacy policy is updated, the changes will be reflected in this document with an updated date. Significant changes will be noted in the application's release notes.
